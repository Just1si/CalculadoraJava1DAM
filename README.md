# Calculadora JavaFX

Aplicación de escritorio desarrollada en **Java** y **JavaFX**, con interfaz gráfica construida mediante **FXML** y una arquitectura sencilla basada en controlador + vista.  
Este proyecto forma parte de mi aprendizaje práctico en desarrollo de interfaces gráficas, eventos y estructura de aplicaciones JavaFX.

## 🚀 Ejecución del proyecto

### 🔧 Requisitos

- **Java 17 o superior**
- **JavaFX SDK 21+**

### ▶️ Ejecutar desde el IDE

Configura los módulos JavaFX en los argumentos de ejecución:
--module-path "ruta/javafx/lib" --add-modules javafx.controls,javafx.fxml


### 💻 Ejecutar en Windows sin IDE

En la sección **Releases** encontrarás un ZIP con:

- `Calculadora.jar`
- `run.bat`

Pasos:

1. Descarga el ZIP desde **Releases**
2. Descomprímelo
3. Doble clic en `run.bat`
4. La calculadora se abrirá automáticamente
